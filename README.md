# Bit Drawers
Bit Drawers is an addon for the Minecraft mods [Chisels &amp; Bits](http://mods.curse.com/mc-mods/minecraft/231095-chisels-bits) by AlgorithmX2 and [Storage Drawers](http://mods.curse.com/mc-mods/minecraft/223852-storage-drawers) by jaquadro. The mod adds special drawes and support for bits.
